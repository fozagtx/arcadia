## AI-Powered UGC Ad Generation Platform

Arcadia helps brands turn ideas into high-converting TikTok ad briefs powered by an agentic workflow and on-chain micropayments. The platform combines AI-driven content generation with blockchain-based compensation systems for creators.

### Key Features

- **AI Brief Generation**: Transform brand ideas into compelling UGC ad briefs using OpenAI API
- **Micropayment System**: x402 protocol integration for efficient creator compensation
- **Blockchain Payments**: Scroll ZK for secure, low-cost on-chain transactions
- **Creator Workflow**: Streamlined process from brief to final ad content
- **Brand Dashboard**: Analytics and campaign management tools

### Tech Stack

- **Frontend**: Next.js 14 with App Router, TypeScript, Tailwind CSS, shadcn/ui
- **AI Integration**: OpenAI API for prompt generation and content optimization
- **Payments**: x402 micropayment protocol with Scroll ZK blockchain
- **Database**: Supabase for user data and campaign management
- **Development**: Modern development workflow with AI-assisted tooling

### Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to see the application.
